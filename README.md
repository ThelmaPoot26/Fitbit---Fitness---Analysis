Fitbit Fitness Analysis
Descripción del proyecto
Proyecto de análisis de datos enfocado en el estudio de la actividad física y los hábitos de sueño de usuarios de dispositivos Fitbit. El objetivo fue identificar patrones de comportamiento relacionados con los pasos diarios, el gasto calórico, la actividad física y el descanso mediante técnicas de análisis de datos y visualización.

Objetivos
-Limpiar y preparar los datos para garantizar su calidad.
-Integrar diferentes conjuntos de datos en una única tabla de análisis.
-Realizar un análisis exploratorio de datos (EDA).
-Identificar relaciones entre actividad física, calorías y sueño.
-Construir un dashboard interactivo para comunicar hallazgos.
-Elaborar un reporte técnico con conclusiones y recomendaciones.

Principales hallazgos
-La mayoría de los usuarios registran entre 4,000 y 12,000 pasos diarios.
-Existe una relación positiva entre la cantidad de pasos y las calorías quemadas.
-La duración del sueño se concentra entre 6 y 8 horas por noche.
-Los días sedentarios son más frecuentes que los altamente activos.
-Un mayor nivel de actividad física no implica necesariamente una mayor duración del sueño.
